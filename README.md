# portfolio-website
A modern portfolio website with a dark/light theme toggle, responsive layout, hero section, social links, and animated images. Built using HTML, CSS, and JavaScript.
# Portfolio Website | Era21Tech

## Features
- Dark/Light mode toggle with smooth switching
- Responsive hero section
- Social media icons and links
- Animated image box hover effects
- Clean and minimal UI design

## Files
- `index.html` — Main structure of the portfolio website.
- `style.css` — Styles for layout, theme, animations, and responsiveness.

## How It Works
- Clicking the theme icon switches between **light** and **dark** themes by toggling a CSS class.
- The hero section displays a name, description, and a "Download CV" button.
- Image hover effects shift the images to create a dynamic look.
- Social icons use Font Awesome.

## How to Use
1. Download or clone the repository.
2. Open `index.html` in any browser.
3. Click the moon/sun icon to change theme.

## Customization
- Replace `images/logo.png`, `girl.png`, `pattern.png` with your own assets.
- Edit text content in `index.html`.
- Update colors, fonts, or animations in `style.css`.
